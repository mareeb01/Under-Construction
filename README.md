# Under Construction

A Coming Soon page with an interactive draggable background.

## Features

- Blurred background with smooth drag scrolling
- Fixed centered card with Coming Soon message
- Responsive design for mobile, tablet, and desktop
- Touch support for mobile devices

## Files

- `index.html` - Main HTML file
- `background.png` - Background image
- `foreground.png` - Coming Soon card image

## Usage

1. Open `index.html` in a web browser
2. Drag the background up and down to scroll
3. The centered card remains fixed in place
